## ✨ Особенности

| | |
|---|---|
| 🖥️ | Красивый Material3 рабочий стол |
| 🎨 | Тёмная / светлая тема |
| 🖼️ | Случайные и свои обои (SVG / PNG / JPG) |
| 🧩 | Установка своих URL-приложений на рабочий стол |
| ⏰ | Точное время на панеле задачи |
| 💽 | **(в тестовой версии)** 7 встроенных приложений |
| 🧠 | Оптимизация — **убрать анимации** |

---

## 📦 Установка

### Через BDI-образ (рекомендуется)

1) Откройте **cmd** в папке где вы скачали **ShOs.**
2) Введите данную **команду:**
```cmd
bdi_installer.bat /ext:ShOs.bdi
```
